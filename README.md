# Hike — Sneaker Store

This project is a demo sneaker e-commerce website called Hike, created for an academic assignment on digital analytics. The site simulates an online shopping experience and demonstrates the implementation of Google Analytics 4 to track user behavior and conversion events.

## Tech Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

```sh
# Install dependencies
npm install

# Start the development server
npm run dev
```

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start dev server on port 8080 |
| `npm run build` | Production build |
| `npm run preview` | Preview production build |
| `npm run test` | Run unit tests |
| `npm run lint` | Run ESLint |
